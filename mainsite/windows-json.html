使用json配置文件自动设置Windows系统的shadowsocks客户端

请严格按照下面的步骤操作，如果有错误请重新仔细对照下面的教程。

1

如果你已经打开了客户端，请先关闭退出再按照下面的步骤操作！

通过客服获取json配置文件

2下载shadowsocks的Windows客户端：

Windows 7/xp：Shadowsocks-win-2.5.6.zip
Windows 8/10：Shadowsocks-win-dotnet4.0-2.5.2.zip

下载并解压保存至上面第一步下载的config-gui文件相同的文件夹内例如桌面，即保证shadowsocks.exe和config-gui.json两个文件在同一个文件夹内。


3打开上面下载解压的Windows客户端shadowsocks.exe，如果有防火墙杀毒软件例如360提示警告之类选择允许即可，然后就可以打开浏览器访问你想访问的网站了。


请删除任何已安装开启的浏览器代理插件例如autoproxy、FoxyProxy、SwitchySharp、红杏等。

当打开shadowsocks客户端后你可以在屏幕右下角（或隐藏的小三角内）找到纸飞机形状的shadowsocks图标，在图标上点右键可弹出菜单，用于更改默认选项：

“服务器选择”选项：可选择不同的服务器连接，前面打勾的是当前连接的服务器，可通过点击其他服务器切换。

“代理模式”选项：
PAC模式：局部代理，只有国外网站才经过代理访问，国内网站不经过代理访问。
全局模式：全局代理，所有网站都经过代理访问。
